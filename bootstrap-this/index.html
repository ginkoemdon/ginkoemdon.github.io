What

In a nutshell, Bootsrap is an open-source collection of reusable pieces of code HTML, CSS, and Javascript

Therefore, this is a time saver, and sweat and tears reducer for front-end developers!

Who

Developed by Twitter Designer/Developers Mark Otto and Jacob Thornton in mid-2010.

Fun fact:
It was originally called Twitter Blueprint

Why

Bootstraps mission is: Making design and development easier.
Behaving as a living style guide and framework that designers & developers can use for free, and without such a big learning-curve.

We will absolutly be using this component the most, with most websites having text! Say goodbye to the days of tediously tweaking your css typhography and a warm HELLO to finally making font our friend.

Headings

  • Bootstrap has a great default typesale of headings from h1-h6.
  • You can either style them with the HTML heading tags, or with a class!
  • Defining headings impacts the font-size, font-weight, and line-height

3. Cards

Geez, I have been waiting for this one. FINALLY! A template for cards instead of having to create them from scratch every time! GLORY!

  • It can hold a wide variety of content such as: headers, footers, images, and more!
  • Cards are built with as little html & css as possible, but still pulls through offering a lot of customability.
  • Built with Flexbox it provides ease of alignment and little spacing utlilities needed.
  • They have no margin by defult, so use spacing utilities when needed.
  • They have no fixed width to start, making them defaulty fill the full width of its parent element- but this is easily customizable with bootstraos sizing options
